迭代器模式（lterator）：在不暴露对象内部结构的同时，可以顺序地访问集合对象内部的各个元素。

实现forEach的迭代器(对低版本不支持的浏览器进行实现)
        // 内部迭代器 forEach，every，map,$.each...
        Array.prototype.forEach = Array.prototype.forEach || function (cb) {
          for (var i = 0; i < this.length; i++) {
            cb(this[i], i, this)
          }
        }
